# AdoptaPerrito - Frontend

## Tecnología
* React 18 + Vite
* JavaScript (Vanilla)

## Requisitos previos
* Node.js >= 18

## Cómo ejecutar
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`
3. Levantar el servidor de desarrollo: `npm run dev`
4. Abrir http://localhost:5173 en el navegador.
